# Cognitive Vision Lab at CVSSP, University of Surrey

Welcome to the GitHub space of the **Cognitive Vision Lab** at the **Centre for Vision, Speech and Signal Processing (CVSSP), University of Surrey**.

Our lab is led by **Prof. Richard Bowden** and focuses on computer vision, machine learning, and multimodal AI, with a strong specialisation in **Sign Language Technology**.
